# ZMK Config

Configuration repo for [ZMK](https://zmkfirmware.dev/docs/).

This will be used to build firmware for my [Cornish-zen](https://lowprokb.ca/products/corne-ish-zen-2)

## Overview

Commit code and let it run, then download artifacts from the actions tab.

For local compilation, use VScode and point the `west` tool at this repo for config.
